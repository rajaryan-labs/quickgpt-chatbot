# QuickGPT Client

This is the frontend application for QuickGPT, a ChatGPT clone.

## Tech Stack

- **React**: UI Library
- **Vite**: Build tool
- **Tailwind CSS**: Styling (To be configured)

## Getting Started

1.  **Install dependencies:**

    ```bash
    npm install
    ```

2.  **Run Development Server:**
    ```bash
    npm run dev
    ```

## Folder Structure

- `src/`: Source code
  - `components/`: Reusable UI components
  - `pages/`: Route pages
  - `assets/`: Static assets
